Implement an \hyperlink{namespaceApache}{Apache} core hook that runs until one of the functions returns something other than ok or decline. That return value is then returned from the hook runner. If the hooks run to completion, then ok is returned. Note that if no hook runs it would probably be more correct to return decline, but this currently does not do so. The implementation is called ap\+\_\+run\+\_\+{\itshape name}.
