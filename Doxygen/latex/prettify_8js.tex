some functions for browser-\/side pretty printing of code contained in html.

For a fairly comprehensive set of languages see the \href{http://google-code-prettify.googlecode.com/svn/trunk/README.html#langs}{\tt R\+E\+A\+D\+ME} file that came with this source. At a minimum, the lexer should work on a number of languages including C and friends, Java, Python, Bash, S\+QL, H\+T\+ML, X\+ML, C\+SS, Javascript, and Makefiles. It works passably on Ruby, P\+HP and Awk and a subset of Perl, but, because of commenting conventions, doesn\textquotesingle{}t work on Smalltalk, Lisp-\/like, or C\+A\+M\+L-\/like languages without an explicit lang class.
